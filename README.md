# Kyo

Kyo is an online marketplace where users can buy and sell items directly to each other. The platform connects individual sellers, small brands, and buyers in one space, offering transparent transactions and secure payments.

## Tech Stack

**Server**: A backend service built with NestJS.

**Client**: A frontend web application developed using Next.js, React.

_Package manager_: `bun`

## Structure

| Codebase |  Description   |
| :------: | :------------: |
| apps/api |   NestJS API   |
| apps/web | Next.js Client |
